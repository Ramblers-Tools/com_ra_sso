# Joomla SSO (Single Sign-On) for Educational Institutes Plugin

Enhance accessibility and streamline authentication with the **Joomla SSO for Educational Institutes** plugin. This powerful solution supports seamless integration with a wide range of **Learning Management Systems (LMS)** such as **ClassLink**, **Clever**, **Canvas**, **Google Classroom**, and more, allowing students and faculty to access all portals using a single set of credentials.

## Key Highlights:
- **Simplified Login Process**:  
  Uses **OAuth/OIDC protocols** to provide seamless access to multiple Joomla sites and LMS portals without repeated logins.
- **Centralized Access Control**:  
  Administrators can define and manage user access levels based on roles, ensuring secure handling of sensitive data.
- **Effortless User Experience**:  
  Ideal for schools, universities, and colleges managing large student populations and digital learning platforms.

## Features:
- **Multisite SSO**:  
  Enable seamless authentication across a network of Joomla multisites.
- **Profile and Attribute Mapping**:  
  Sync user profiles (students, teachers, staff) from the OAuth Server.
- **Group-Based Content & Subsite Access**:  
  Control access to resources and subsites based on OAuth server group membership.
- **User Provisioning & Deprovisioning**:  
  Automate real-time user creation, updates, and deletion.
- **Multiple OAuth Provider Support**:  
  Integrates with **Shibboleth**, **Azure AD**, **Google Workspace**, and more.
- **Joomla LMS Support**:  
  Compatible with Joomla LMS platforms like **Learndash**, **LearnPress**, **TutorLMS**, etc.

## Why Choose This Plugin?
- **Security & Efficiency**:  
  Centralized, secure access to multiple systems with a single login.
- **Improved Workflow**:  
  Eliminates the need to remember multiple usernames and passwords.
- **Customizable Solutions**:  
  Supports advanced features like **Attribute-Based Redirection** and **Session Management**.

## Add-Ons:
- **Keycloak User Sync**: Sync user data between Joomla and Keycloak.
- **SCIM User Provisioning**: Enable real-time user updates using SCIM protocols.
- **Role/Group-Based Redirection**: Redirect users based on OAuth server roles/groups.
- **SSO Login Audit**: Track user activity and generate detailed reports.
- **REST API Authentication**: Secure Joomla pages and sites using REST API authentication.

## Installation
- Download the latest installable package from: [OIDC / OAuth Client Plugin](https://plugins.miniorange.com/joomla-single-sign-on-sso-oauth-oidc) (Recommended)  
OR
- Go to the **Releases** folder of this repository. Download the file: `miniorange-sso-for-educational-institutes.zip`

## Quick Setup
1. Log in to your Joomla Administrator panel.
2. Navigate to:  **Extensions → Manage → Install**
3. Install the plugin using one of the following methods:
   - Upload the downloaded .zip file  
    OR 
   - Search for **miniOrange OAuth** plugin and install it directly.
4. After successful installation, click **Get Started**.
5. Follow the [Setup Guide](https://plugins.miniorange.com/joomla-sso-ldap-mfa-solutions?section=oauth-client) to configure the plugin.
