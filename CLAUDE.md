# com_ra_sso

Vendored/customized Joomla OAuth/OIDC SSO login plugin (based on the MiniOrange
`miniorange-joomla-oauth-client-free-plugin`), used to provide SSO login for
Ramblers Tools Joomla sites.

Vault notes: `~/vault/ra-sso/`
Knowledge graph: `~/vault/ra-sso/graph/`

See `~/vault/CLAUDE.md` for the full vault workflow (`/vault-resume`, `/vault-save`,
Zettelkasten rules, graphify integration).
