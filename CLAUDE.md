# ra-sso

Joomla OAuth/OIDC SSO login plugin used to provide SSO login for Ramblers Tools
Joomla sites. Originally forked from MiniOrange's `miniorange-joomla-oauth-client-free-plugin`
but has since been rebranded and de-vendored into a standalone product.

Vault notes: `~/vault/ra-sso/`
Knowledge graph: `~/vault/ra-sso/graph/`

See `~/vault/CLAUDE.md` for the full vault workflow (`/vault-resume`, `/vault-save`,
Zettelkasten rules, graphify integration).
