# Azure AD Single Sign-On (OAuth / OpenID Connect SSO) for Joomla

Simplify authentication and user access with **Azure AD Single Sign-On (OAuth OpenID Connect SSO)** for Joomla. This robust plugin enables seamless integration with **Azure AD**, **Azure AD B2C**, and **Office 365**, allowing users to log in securely using their Microsoft account credentials.

## Key Highlights:
- **Unlimited SSO**:  
  Enable Single Sign-On across multiple Joomla domains and Microsoft applications (Azure AD, Azure AD B2C, Office 365).
- **Flexible Integration**:  
  Works with **OAuth 2.0** and **OpenID Connect (OIDC)** protocols for scalable and secure authentication.
- **Advanced Profile Syncing**:  
  Sync Azure user profiles (given name, family name, email, and custom attributes) with Joomla.

## Supported Grant Types:
- Authorization Code Grant  
- Implicit Grant  
- Resource Owner Password Credentials Grant  
- Client Credentials Grant  
- Refresh Token Grant  
- Hybrid Grant  
- PKCE Flow (Proof Key for Code Exchange)

## Versions & Features:

### **Free Version**
- Auto user registration
- Basic attribute mapping (Username, Email)
- Limited authentication

### **Standard Version**
- Unlimited authentication and user registration
- Login widget customization
- Custom redirect URLs
- Basic group mapping

### **Premium Version**
- Advanced attribute and group mapping
- Force authentication for secure access
- OpenID Connect support
- JWT validation
- Domain-specific registration

### **Enterprise Version**
- Includes all Premium features
- Grant settings and endpoint configurations
- Login analytics and reporting

## Add-Ons:
- **Azure User Sync**: Sync and provision users between Joomla and Azure AD.
- **Role/Group-Based Redirection**: Redirect users based on roles/groups received from Azure.
- **SSO Session Management**: Manage login session duration.
- **Login Audit**: Capture and generate reports for SSO user activity.
- **REST API Authentication**: Secure Joomla REST APIs using OAuth.

## Use Cases:
- Login with **Azure AD**, **Azure AD B2C**, or **Office 365**.
- Custom login redirects using sign-up, sign-in, or lost password policies.
- Social login integrations such as **Facebook** and **Google**.

---

## Quick Setup
1. Log in to your Joomla Administrator panel.
2. Navigate to:  **Extensions → Manage → Install**
3. Install the plugin using one of the following methods:
   - Upload the downloaded .zip file  
    OR 
   - Search for **miniOrange OAuth** plugin and install it directly.
4. After successful installation, click **Get Started**.
5. Follow the [Setup Guide](https://plugins.miniorange.com/joomla-sso-ldap-mfa-solutions?section=oauth-client) to configure the plugin.
