# Joomla OAuth / OIDC Single Sign-On (SSO) Plugin

Securely log into your Joomla site using **miniOrange’s OAuth/OIDC Single Sign-On (SSO) Plugin**. This plugin enables seamless integration with various **OAuth 2.0** and **OpenID Connect (OIDC)** providers for effortless and secure authentication.

## Key Highlights:
- **Multiple Provider Support**:  
  Connect with OAuth 2.0 and OIDC providers such as **Azure**, **AWS Cognito**, **Keycloak**, and more.
- **Limitless Authentication**:  
  Enjoy unlimited user logins using OAuth and OIDC protocols.
- **Profile & Role Mapping**:  
  Easily map user attributes and roles to Joomla user profiles.
- **Wide Grant Type Support**:  
  Supports **Authorization Code**, **Implicit**, **Refresh Token**, **Hybrid**, and **PKCE** flows.

## Versions & Features:

### **Free Version**
- Basic authentication with third-party OAuth and custom servers
- Basic profile mapping and limited user registration
- Supports multiple languages

### **Standard Version**
- Unlimited user registration and authentication
- Automatic user registration and widget customization
- Basic group mapping and redirect URL support

### **Premium Version**
- Advanced attribute and group mapping
- OpenID Connect support, **JWT validation**, and forced authentication
- Comprehensive language support

### **Enterprise Version**
- Multi-provider support and access token retrieval for API calls
- Login reports and analytics
- Additional endpoint configuration options

## Popular Use Cases:
- **Azure SSO**: Enable SSO with **Azure AD**, **Azure AD B2C**, and **Office 365**.
- **AWS Cognito**: Sync user profiles with roles for secure logins.
- **Discord SSO**: Synchronize profiles and roles for Discord-based access.
- **Keycloak Integration**: Role-based access control with token-based external calls.

## Installation
- Download the latest installable package from: [OIDC / OAuth Client Plugin](https://plugins.miniorange.com/joomla-single-sign-on-sso-oauth-oidc) (Recommended)  
OR
- Go to the **Releases** folder of this repository. Download the file: `Login+with+Azure+AD.zip`

---

## Quick Setup
1. Log in to your Joomla Administrator panel.
2. Navigate to:  **Extensions → Manage → Install**
3. Install the plugin using one of the following methods:
   - Upload the downloaded .zip file  
    OR 
   - Search for **miniOrange OAuth** plugin and install it directly.
4. After successful installation, click **Get Started**.
5. Follow the [Setup Guide](https://plugins.miniorange.com/joomla-sso-ldap-mfa-solutions?section=oauth-client) to configure the plugin.
